Comparison of Movies on Streaming Services

The Grid Chart is divided into 3x3 bands with segmented financial performance as the measurement on the y-axis and segmented rotten tomatoes score as the measurement on the x-axis. Segmented Financial Performance: - "Financial Performance" is calculated by taking the log base e of each movie's Gross Revenue/Cost. - Each row of the y-axis (LOW, MED, HIGH) contains an equal range of Financial Performance, which is dynamically generated based on the selected data. Segmented Rotten Tomatoes Score: - Each column of the x-axis (LOW, MED, HIGH) contains an equal range of Rotten Tomatoes Score, which is statically set at [0, 100]. Each grid contains a pie chart showing the proportion of movies by platform for the intersection of the appropriate Financial Performance and Rotten Tomatoes Score. The size of each pie chart is a rough indicator of the total number of movies that are within each grid. A larger pie chart represents a greater number of movies while a smaller pie chart represents a smaller number of movies. If there is nothing in a grid, this means that there are not any movies that fit within these measurements.
