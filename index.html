Comparison of Movies on Streaming Services

The Grid Chart to the left (Proportion of Movies by Platform, Financial Performance, and Rotten Tomatoes Score) is divided into 3x3 bands.

The size of each pie chart is an indicator of the total number of movies in each grid. A larger pie chart represents a greater number of movies. If there is nothing in a grid, there are not any movies that fit within these measurements.

Segmented Financial Performance (Y-Axis):

"Financial Performance" is calculated by taking the log base e of each movie's Gross Revenue/Cost.

Each row of the y-axis (LOW, MED, HIGH) contains an equal range of Financial Performance, which is dynamically generated based on the selected data.

Segmented Rotten Tomatoes Score (X-Axis):

Each column of the x-axis (LOW, MED, HIGH) contains an equal range of Rotten Tomatoes Score, which is statically set at [0, 100].

Each grid contains a pie chart showing the proportion of movies by platform for the intersection of the appropriate Financial Performance and Rotten Tomatoes Score.
